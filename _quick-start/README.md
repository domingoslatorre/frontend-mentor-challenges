# Quick start

## How to use

1. `npm install`
1. Edit files in `/src`
1. `npm run`
1. Gulp will watch/run the tasks and generate the results in `/public`

## Tools
- Gulp as task automation tool
- SASS as CSS pre-processor
- PostCSS
  - autoprefixer
  - cssnano
- Browserify and babelify to write ES6+ features
- Pug as template engine for HTML
